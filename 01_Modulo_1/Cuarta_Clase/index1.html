Primer ejercicio con HTML

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).

  1. Es un lenguaje de etiquetas que indica al navegador cómo organizar el contenido
  2. Es fácil de aprender y usar
  3. Permite crear hipertextos, es decir, textos que se pueden hacer clic para navegar por internet
  4. Se puede agregar imágenes, videos y audio a las páginas web
  5. Es un lenguaje de código abierto y gratuito
  6. Tiene una sintaxis uniforme y estructurada
